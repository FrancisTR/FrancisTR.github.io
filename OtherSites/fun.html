More About Me!

I am passionate in creating games. Before taking the Game Design course during my Junior year, I self-taught on using the Godot engine and exploring what the engine has to offer. Not only I got a grasp on the concepts of using Godot, but it prepared me to be successful in the Game Design class since I was able to transfer my skills onto learning a new game engine, which was Gamemaker.

Fast forward to my Senior year, I have co-hosted a Game Jam event for the University of St. Thomas. I was responsible for creating the event using itch.io and advertising the event to the Computer Science, Digital Media Arts, and English department. Although there were a lot of tasks that weighed on me, I was able to manage both as a co-host and participant and our team was able to create a quality game under one week.
